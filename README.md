# My Portfolio

This is the source code for my portfolio.

## Running the code

1. Run `npm i` to install the dependencies.
2. Run `npm run dev` to start the development server.
3. Open `http://localhost:5173` in your browser.

## Features
-   **Resend Integration**: Fully functional contact form.
-   **Premium UI**: Custom hover effects and animations using Framer Motion.
-   **Form Validation**: Client-side validation for Name, Email, and Message.
